# 2.4G 模块配置需求

## 1. 目标

APP 固件需要支持 2.4G 模块点对点配置。程序下载后首次启动时自动进入模块配置流程；正常运行时，USART2 收到 `+++` 后进入手动配置模式。进入配置模式后，应支持通过 `AT&V` 查看模块当前配置。

## 2. 配置命令分类

### 主站配置

主站 ID 为 `1`，从站 ID 为 `2`，网络 ID 为 `1234567890`。

```text
ATS133=1
ATS101=0
ATS105=1
ATS140=2
ATS104=1234567890
ATS102=1
ATS113=1
AT&W
```

### 从站配置

主站 ID 为 `1`，从站 ID 为 `2`，网络 ID 为 `1234567890`。

```text
ATS133=1
ATS101=2
ATS105=2
ATS140=1
ATS104=1234567890
ATS102=1
ATS113=1
AT&W
```

### 查询命令

```text
AT&V
```

进入配置模式后，`AT&V` 用于查看当前配置。首次自动配置流程在写入并保存配置后，也会再次发送 `AT&V` 查询确认。

## 3. 命令发送规则

- `+++` 是进入命令模式的逃逸序列，发送时不追加回车。
- 发送 `+++` 后需要等待模块返回应答信息。
- 后续所有 `AT...` 命令发送时需要追加回车 `\r`。
- 每发送完一条配置命令，都需要等待模块返回 `ACK` 或 `OK`。
- 进入配置模式时应自动扫描常见模块波特率：`9600`、`19200`、`38400`、`57600`、`115200`。

## 4. 首次启动配置流程

1. APP 启动后检查 Flash 配置标记。
2. 如果标记不存在，等待模块上电稳定。
3. 发送不带回车的 `+++`，尝试进入 AT 命令模式。
4. 如果当前波特率失败，则切换到备用波特率再次尝试。
5. 发送带回车的 `AT&V` 查看当前配置。
6. 按设备角色发送主站或从站配置命令，每条命令都等待 `ACK` 或 `OK`。
7. 发送带回车的 `AT&W` 保存配置，并等待 `ACK` 或 `OK`。
8. 再次发送带回车的 `AT&V` 查询确认。
9. 发送带回车的 `ATO` 退出命令模式。
10. 写入 Flash 配置标记，后续普通复位不重复自动配置。

重新下载 APP 固件会擦除应用区尾部标记页，因此下载后的第一次启动会重新执行配置流程。

## 5. 手动配置流程

1. 正常运行时监听 USART2。
2. 当 USART2 收到连续 `+++` 时，进入模块 AT 命令模式。
3. `+++` 不转发给 USART1，也不追加回车。
4. 进入后自动发送带回车的 `AT&V`，并把模块返回内容转发到 USART2。
5. 配置模式中，USART2 与 USART1 直通。
6. 用户后续输入 AT 命令时，需要以回车结束。
7. 空闲超过配置超时时间后自动退出手动配置直通状态。

## 6. 串口透传

正常运行时只启用 USART1 和 USART2 的双向透传：

- USART2 收到的数据转发到 USART1。
- USART1 收到的数据转发到 USART2。
- USART2 收到 `+++` 时不转发给 USART1，而是进入配置模式。
- USART3 当前不参与透传链路。

## 7. 固件配置项

当前默认编译为从站：

```c
#define APP_RADIO_DEVICE_ROLE APP_RADIO_ROLE_SLAVE
#define APP_RADIO_LOCAL_ID    2U
#define APP_RADIO_REMOTE_ID   1U
#define APP_RADIO_NETWORK_ID  "1234567890"
```

如果要编译主站，将配置改为：

```c
#define APP_RADIO_DEVICE_ROLE APP_RADIO_ROLE_MASTER
#define APP_RADIO_LOCAL_ID    1U
#define APP_RADIO_REMOTE_ID   2U
```

## 8. 验收标准

- APP 可编译生成 `build/application/application.hex`。
- 下载 APP 后首次启动会执行 AT 配置流程。
- `+++` 发送时不追加回车。
- 后续 AT 命令发送时追加回车。
- 每条配置命令发送后等待模块返回 `ACK` 或 `OK`。
- 手动发送 `+++` 能进入配置模式。
- 配置模式中发送 `AT&V` 能看到模块当前配置。
- 正常运行时 USART2 数据透传到 USART1，USART1 数据透传到 USART2。
- 模块默认波特率按 `9600` 进入配置。
- `ATS102=1` 后 APP 和模块运行波特率为 `115200`。
- `AT&W` 保存后，普通复位不会重复自动配置。
